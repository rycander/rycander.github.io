RECENT PROJECTS

AGGREDILE

Minimalistic Feedly clone. Allows the user to view, follow and unfollow any rss feed, and aggregates all followed feeds to a single list. Made with Rails, Bootstrap, and Backbone.js.

ASTEROIDS AND LASERS

Asteroids clone with some minor differences, written in Javascript. Fly around with WASD keys, and fire with the space key. Instead of firing bullets, the player fires a giant laser that sends them flying backwards.

SONAR

Video game in which the player controls a blind protagonist, and has to create areas of vision through 'echolocation' by making sounds into their microphone. Final project in college, made with a team, using C# and XNA.

Conway's Game of Life

Implementation of Conway's Game of Life, written entirely with Javascript and HTML5 canvas in 48 hours for a coding challenge.
